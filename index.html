    URL Google Apps Script (GAS)

    Google Apps Script berfungsi sebagai backend/server untuk menyimpan dan mengambil data absensi sekolah ke Google Sheet.

    Cara Mendapatkan URL GAS:

    1. Buka Google Sheets baru atau yang sudah ada
    2. Klik menu Extensions → Apps Script
    3. Copy-paste kode code.gs yang disediakan developer ke editor
    4. Klik Deploy → New deployment
    5. Pilih type Web app, set "Execute as" ke Me, "Who has access" ke Anyone
    6. Klik Deploy, lalu Copy URL Web app yang muncul
    7. Paste URL tersebut di kolom di bawah ini

    URL biasanya berformat: https://script.google.com/macros/s/[ID]/exec

    2

    URL Google Sheet (Opsional)

    URL Google Sheet utama untuk melihat data langsung di spreadsheet. Juga digunakan untuk cetak kartu barcode.

    Cara Mendapatkan URL Sheet:

    1. Buka Google Sheets yang sama dengan Apps Script di atas
    2. Di bar alamat browser, copy URL spreadsheet tersebut
    3. Paste URL tersebut di kolom di bawah ini
    4. URL biasanya berformat: https://docs.google.com/spreadsheets/d/[ID]/edit

    URL Google Sheet untuk melihat data secara langsung dan cetak kartu barcode

    Informasi Penting

    Setiap sekolah menggunakan Google Sheet masing-masing. Data tidak tercampur antar sekolah.

    Google Sheet gratis — tidak ada biaya server. Semua data tersimpan aman di cloud Google.

    Apps Script berfungsi sebagai jembatan antara aplikasi dan Google Sheet.

    Hubungi developer jika membutuhkan bantuan setup Google Apps Script.
